# Hola mundo

Este blog pretende ser una colleción, ordenada temporalmente, de algunas de las ideas que, por algún motivo, circulan por mi cabeza.
En otros tiempos este tipo de cosas las escribía en las redes sociales, pero el ruído acabó molestándome demasiado.
Habrá entradas tanto en castellano como en inglés, así que, que nadie espere demasiado orden.
