# Courses ## Computing - [Programming 101 in Python](https://bioinfcomav.github.io/courses/intro_programming). A interactive web course on the most essential programming ideas in Python. - [Programming paradigms](https://bioinfcomav.github.io/courses/programming_paradigms). A discussion about procedural, object oriented and functional programming with practical examples in Python. - Python [project management](https://github.com/JoseBlanca/uv_seminar) with uv. - Introduction to [Linux](https://bioinfcomav.github.io/courses/unix/index.html). An introduction to the Unix command line (in Spanish). - Python for the data analysis, [slides](https://drive.google.com/drive/folders/1RHV-Gni3eDTZHiiLzJPQPYrJdbVuuEbt?usp=drive_link) and github [repository](https://github.com/JoseBlanca/py_industriales). ## Bioinformatics - Practical introduction to [bioinformatics](https://bioinfcomav.github.io/courses/biotech3/index.html). A course focused on learning the basics about sequence databases, sequence alimengts, blast, and phylogenetics. - Introduction to [NGS sequence analysis](https://bioinfcomav.github.io/courses/sequence_analysis/index.html). Sequence quality, assembly, mapping and SNP calling. ## Genetics - [Plant genomics](https://drive.google.com/drive/folders/1yOTHXimatswiwm8kwGPTa1oR9ATbMZh3?usp=sharing) for the breeder. - [Population genetics](https://drive.google.com/drive/folders/1k9uZpbrz4cNGw1vEzpWaHUD3-fpUxpNT?usp=drive_link) for the breeder. ## Bioethics - [Bioethics 101](https://drive.google.com/drive/folders/100WJpcPj30_456OczrIyHI6vFvIV6T82?usp=sharing) - Seminario sobre [agricultura y moral](https://drive.google.com/drive/folders/1XYM2AvF9a_ObQXRDbbiljoGlTu24JjRh?usp=sharing)
